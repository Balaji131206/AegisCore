Welcome to the documentation for the MultiThreadSystemJAVA project.

This site is powered by Docusaurus and serves as the central place to maintain documentation for every step of the project.

## Getting started

- Run the Java server from `src/`
- Use the existing `docs/` directory for project documentation
- Open the site with `npm start` from the `website/` folder
